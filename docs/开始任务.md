
### 任务目标

这是本项目的基本目标，基于这些目标完善openspec/project.md文件

1. 文件浏览器
2. 跨平台
3. 支持文件操作（复制、移动、删除、重命名等）
4. 使用 rust 的 workspaces 管理多个模块（app，comps 等）
5. 使用 gpui 构建图形界面，组件需要自行实现
6. 支持多标签页浏览
7. 支持分栏浏览，支持横向和纵向分栏
8. 项目分为左右布局，左侧布局显示磁盘分区和常用目录，右侧布局显示当前目录内容
